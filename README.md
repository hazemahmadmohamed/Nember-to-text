# Number To Text Converter (C++)

This C++ program converts numbers into English words.  
It supports values from 1 up to billions (999,999,999,999).

## Features
- Handles numbers in thousands, millions, and billions.
- Works with integers up to 12 digits.
- Simple recursive implementation.

## Example
Input: 999999999
Output: Nine Hundred Ninety-Nine Million Nine Hundred Ninety-Nine Thousand Nine Hundred Ninety-Nine

_____________________________________________________________________________________________________________
C++ program to convert numbers into English words

Examples:
1 → One

## How to Run
1. Compile the file using Visual Studio or g++.
2. Run the executable.
3. Enter a number to see its text form.
